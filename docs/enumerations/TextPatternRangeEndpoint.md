# TextPatternRangeEndpoint enumeration (uiautomationcore.h)

Contains values that specify the endpoints of a text range.

## Syntax

```
typedef enum TextPatternRangeEndpoint {
  TextPatternRangeEndpoint_Start = 0,
  TextPatternRangeEndpoint_End = 1
} ;
```

## Constants

|  |
| --- |
| `TextPatternRangeEndpoint_Start` Value: *0* The starting endpoint of the range. |
| `TextPatternRangeEndpoint_End` Value: *1* The ending endpoint of the range. |
