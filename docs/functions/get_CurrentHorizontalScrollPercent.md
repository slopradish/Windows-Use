# IUIAutomationScrollPattern::get\_CurrentHorizontalScrollPercent method (uiautomationclient.h)

Retrieves the horizontal scroll position.

This property is read-only.

## Syntax

```
HRESULT get_CurrentHorizontalScrollPercent(
  double *retVal
);
```

## Parameters

`retVal`

## Return value

None
