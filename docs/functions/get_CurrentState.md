# IUIAutomationLegacyIAccessiblePattern::get\_CurrentState method (uiautomationclient.h)

Retrieves the Microsoft Active Accessibility state identifier for the element.

This property is read-only.

## Syntax

```
HRESULT get_CurrentState(
  DWORD *pdwState
);
```

## Parameters

`pdwState`

## Return value

None
