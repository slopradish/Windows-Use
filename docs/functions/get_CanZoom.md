# ITransformProvider2::get\_CanZoom method (uiautomationcore.h)

Indicates whether the control supports zooming of its viewport.

This property is read-only.

## Syntax

```
HRESULT get_CanZoom(
  BOOL *pRetVal
);
```

## Parameters

`pRetVal`

## Return value

None
