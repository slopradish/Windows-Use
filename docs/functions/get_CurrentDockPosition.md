# IUIAutomationDockPattern::get\_CurrentDockPosition method (uiautomationclient.h)

Retrieves the dock position of this element within its docking container.

This property is read-only.

## Syntax

```
HRESULT get_CurrentDockPosition(
  DockPosition *retVal
);
```

## Parameters

`retVal`

## Return value

None
