# ISpreadsheetItemProvider::get\_Formula method (uiautomationcore.h)

Specifies the formula for this spreadsheet cell.

This property is read-only.

## Syntax

```
HRESULT get_Formula(
  BSTR *pRetVal
);
```

## Parameters

`pRetVal`
