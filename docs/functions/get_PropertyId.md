# IUIAutomationPropertyCondition::get\_PropertyId method (uiautomationclient.h)

Retrieves the identifier of the property on which this condition is based.

This property is read-only.

## Syntax

```
HRESULT get_PropertyId(
  PROPERTYID *propertyId
);
```

## Parameters

`propertyId`

## Return value

None
