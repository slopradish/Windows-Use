# IUIAutomationElement::get\_CurrentHasKeyboardFocus method (uiautomationclient.h)

Indicates whether the element has keyboard focus.

This property is read-only.

## Syntax

```
HRESULT get_CurrentHasKeyboardFocus(
  BOOL *retVal
);
```

## Parameters

`retVal`

## Return value

None
