# IUIAutomationSelectionItemPattern::get\_CachedIsSelected method (uiautomationclient.h)

A cached value that indicates whether this item is selected.

This property is read-only.

## Syntax

```
HRESULT get_CachedIsSelected(
  BOOL *retVal
);
```

## Parameters

`retVal`

## Return value

None
