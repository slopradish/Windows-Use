# ITransformProvider::get\_CanMove method (uiautomationcore.h)

Indicates whether the control can be moved.

This property is read-only.

## Syntax

```
HRESULT get_CanMove(
  BOOL *pRetVal
);
```

## Parameters

`pRetVal`

## Return value

None
