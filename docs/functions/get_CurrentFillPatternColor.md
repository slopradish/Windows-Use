# IUIAutomationStylesPattern::get\_CurrentFillPatternColor method (uiautomationclient.h)

Retrieves the color of the pattern used to fill an element in a document.

This property is read-only.

## Syntax

```
HRESULT get_CurrentFillPatternColor(
  int *retVal
);
```

## Parameters

`retVal`

## Return value

None
