# IUIAutomationTransformPattern2::get\_CurrentCanZoom method (uiautomationclient.h)

Indicates whether the control supports zooming of its viewport.

This property is read-only.

## Syntax

```
HRESULT get_CurrentCanZoom(
  BOOL *retVal
);
```

## Parameters

`retVal`

## Return value

None
