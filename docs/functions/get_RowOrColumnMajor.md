# ITableProvider::get\_RowOrColumnMajor method (uiautomationcore.h)

Specifies the primary direction of traversal for the table.

This property is read-only.

## Syntax

```
HRESULT get_RowOrColumnMajor(
  RowOrColumnMajor *pRetVal
);
```

## Parameters

`pRetVal`

## Return value

None
