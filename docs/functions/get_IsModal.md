# IWindowProvider::get\_IsModal method (uiautomationcore.h)

Indicates whether the window is modal.

This property is read-only.

## Syntax

```
HRESULT get_IsModal(
  BOOL *pRetVal
);
```

## Parameters

`pRetVal`

## Return value

None
