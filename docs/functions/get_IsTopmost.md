# IWindowProvider::get\_IsTopmost method (uiautomationcore.h)

Indicates whether the window is the topmost element in the z-order.

This property is read-only.

## Syntax

```
HRESULT get_IsTopmost(
  BOOL *pRetVal
);
```

## Parameters

`pRetVal`

## Return value

None
