# IUIAutomationStylesPattern::get\_CurrentExtendedProperties method (uiautomationclient.h)

Retrieves a localized string that contains the list of extended properties for an element in a document.

This property is read-only.

## Syntax

```
HRESULT get_CurrentExtendedProperties(
  BSTR *retVal
);
```

## Parameters

`retVal`

## Return value

None
