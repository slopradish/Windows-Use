# IUIAutomationProxyFactoryMapping::get\_Count method (uiautomationclient.h)

Retrieves the number of entries in the proxy factory table.

This property is read-only.

## Syntax

```
HRESULT get_Count(
  UINT *count
);
```

## Parameters

`count`

## Return value

None
