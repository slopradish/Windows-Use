# IUIAutomationTransformPattern::get\_CachedCanResize method (uiautomationclient.h)

Retrieves a cached value that indicates whether the element can be resized.

This property is read-only.

## Syntax

```
HRESULT get_CachedCanResize(
  BOOL *retVal
);
```

## Parameters

`retVal`

## Return value

None
