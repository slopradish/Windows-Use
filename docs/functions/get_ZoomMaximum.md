# ITransformProvider2::get\_ZoomMaximum method (uiautomationcore.h)

Retrieves the maximum zoom level of the element.

This property is read-only.

## Syntax

```
HRESULT get_ZoomMaximum(
  double *pRetVal
);
```

## Parameters

`pRetVal`

## Return value

None
