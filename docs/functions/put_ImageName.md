# IUIAutomationProxyFactoryEntry::put\_ImageName method (uiautomationclient.h)

Sets or retrieves the name of the image of the proxy factory.

This property is read/write.

## Syntax

```
HRESULT put_ImageName(
  LPCWSTR imageName
);
```

## Parameters

`imageName`

## Return value

None
