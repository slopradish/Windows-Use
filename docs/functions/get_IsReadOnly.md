# IRangeValueProvider::get\_IsReadOnly method (uiautomationcore.h)

Indicates whether the value of a control is read-only.

This property is read-only.

## Syntax

```
HRESULT get_IsReadOnly(
  BOOL *pRetVal
);
```

## Parameters

`pRetVal`

## Return value

None
