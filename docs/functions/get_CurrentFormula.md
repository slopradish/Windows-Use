# IUIAutomationSpreadsheetItemPattern::get\_CurrentFormula method (uiautomationclient.h)

Retrieves the formula for this cell.

This property is read-only.

## Syntax

```
HRESULT get_CurrentFormula(
  BSTR *retVal
);
```

## Parameters

`retVal`

## Return value

None
