# IUIAutomationLegacyIAccessiblePattern::get\_CurrentRole method (uiautomationclient.h)

Retrieves the Microsoft Active Accessibility role identifier of the element.

This property is read-only.

## Syntax

```
HRESULT get_CurrentRole(
  DWORD *pdwRole
);
```

## Parameters

`pdwRole`

## Return value

None
