# ISelectionItemProvider::get\_IsSelected method (uiautomationcore.h)

Indicates whether an item is selected.

This property is read-only.

## Syntax

```
HRESULT get_IsSelected(
  BOOL *pRetVal
```

## Parameters

`pRetVal`
