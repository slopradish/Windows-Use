# IUIAutomationLegacyIAccessiblePattern::get\_CachedName method (uiautomationclient.h)

Retrieves the cached Microsoft Active Accessibility name property of the element.

This property is read-only.

## Syntax

```
HRESULT get_CachedName(
  BSTR *pszName
);
```

## Parameters

`pszName`

## Return value

None
