# IUIAutomationAnnotationPattern::get\_CurrentAnnotationTypeId method (uiautomationclient.h)

Retrieves a value that identifies the annotation's type.

This property is read-only.

## Syntax

```
HRESULT get_CurrentAnnotationTypeId(
  int *retVal
);
```

## Parameters

`retVal`

## Return value

None
