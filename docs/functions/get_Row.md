# IGridItemProvider::get\_Row method (uiautomationcore.h)

Specifies the ordinal number of the row that contains this cell or item.

This property is read-only.

## Syntax

```
HRESULT get_Row(
  int *pRetVal
);
```

## Parameters

`pRetVal`

## Return value

None
