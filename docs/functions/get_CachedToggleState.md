# IUIAutomationTogglePattern::get\_CachedToggleState method (uiautomationclient.h)

Retrieves the cached state of the control.

This property is read-only.

## Syntax

```
HRESULT get_CachedToggleState(
  ToggleState *retVal
);
```

## Parameters

`retVal`

## Return value

None
