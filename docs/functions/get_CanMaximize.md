# IWindowProvider::get\_CanMaximize method (uiautomationcore.h)

Indicates whether the window can be maximized.

This property is read-only.

## Syntax

```
HRESULT get_CanMaximize(
  BOOL *pRetVal
);
```

## Parameters

`pRetVal`

## Return value

None
