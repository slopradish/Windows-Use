# IUIAutomationSelectionPattern2::get\_CurrentItemCount method (uiautomationclient.h)

Gets an integer value indicating the number of selected items.

This property is read-only.

## Syntax

```
HRESULT get_CurrentItemCount(
  int *retVal
);
```

## Parameters

`retVal`

## Return value

None
