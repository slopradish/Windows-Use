# IUIAutomationTablePattern::get\_CachedRowOrColumnMajor method (uiautomationclient.h)

Retrieves the cached primary direction of traversal for the table.

This property is read-only.

## Syntax

```
HRESULT get_CachedRowOrColumnMajor(
  RowOrColumnMajor *retVal
);
```

## Parameters

`retVal`

## Return value

None
