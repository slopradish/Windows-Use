# IUIAutomationGridItemPattern::get\_CachedRowSpan method (uiautomationclient.h)

Retrieves the cached number of rows spanned by a grid item.

This property is read-only.

## Syntax

```
HRESULT get_CachedRowSpan(
  int *retVal
);
```

## Parameters

`retVal`

## Return value

None
