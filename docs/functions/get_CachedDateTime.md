# IUIAutomationAnnotationPattern::get\_CachedDateTime method (uiautomationclient.h)

Retrieves the cached date and time that this annotation was created.

This property is read-only.

## Syntax

```
HRESULT get_CachedDateTime(
  BSTR *retVal
);
```

## Parameters

`retVal`

## Return value

None
