# IUIAutomationTransformPattern2::get\_CurrentZoomMinimum method (uiautomationclient.h)

Retrieves the minimum zoom level of the control's viewport.

This property is read-only.

## Syntax

```
HRESULT get_CurrentZoomMinimum(
  double *retVal
);
```

## Parameters

`retVal`

## Return value

None
