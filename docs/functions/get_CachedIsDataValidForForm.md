# IUIAutomationElement::get\_CachedIsDataValidForForm method (uiautomationclient.h)

Retrieves a cached value that indicates whether the element contains valid data for the form.

This property is read-only.

## Syntax

```
HRESULT get_CachedIsDataValidForForm(
  BOOL *retVal
);
```

## Parameters

`retVal`

## Return value

None
