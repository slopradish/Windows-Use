# IUIAutomationProxyFactoryEntry::get\_AllowSubstringMatch method (uiautomationclient.h)

Sets or retrieves a value that specifies whether the proxy allows substring matching.

This property is read/write.

## Syntax

```
HRESULT get_AllowSubstringMatch(
  BOOL *allowSubstringMatch
);
```

## Parameters

`allowSubstringMatch`

## Return value

None
