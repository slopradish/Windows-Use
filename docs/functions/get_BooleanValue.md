# IUIAutomationBoolCondition::get\_BooleanValue method (uiautomationclient.h)

Retrieves the value of the condition: either **TRUE** or **FALSE**.

This property is read-only.

## Syntax

```
HRESULT get_BooleanValue(
  BOOL *boolVal
);
```

## Parameters

`boolVal`

## Return value

None
