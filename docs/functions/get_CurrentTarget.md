# IUIAutomationAnnotationPattern::get\_CurrentTarget method (uiautomationclient.h)

Retrieves the element that is being annotated.

This property is read-only.

## Syntax

```
HRESULT get_CurrentTarget(
  IUIAutomationElement **retVal
);
```

## Parameters

`retVal`

## Return value

None
