# IExpandCollapseProvider::get\_ExpandCollapseState method (uiautomationcore.h)

Indicates the state, expanded or collapsed, of the control.

This property is read-only.

## Syntax

```
HRESULT get_ExpandCollapseState(
  ExpandCollapseState *pRetVal
);
```

## Parameters

`pRetVal`

## Return value

None
