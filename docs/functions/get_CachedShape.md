# IUIAutomationStylesPattern::get\_CachedShape method (uiautomationclient.h)

Retrieves the cached shape of an element in a document.

This property is read-only.

## Syntax

```
HRESULT get_CachedShape(
  BSTR *retVal
);
```

## Parameters

`retVal`

## Return value

None
