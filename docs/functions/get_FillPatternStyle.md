# IStylesProvider::get\_FillPatternStyle method (uiautomationcore.h)

Specifies the fill pattern style of an element in a document.

This property is read-only.

## Syntax

```
HRESULT get_FillPatternStyle(
  BSTR *retVal
);
```

## Parameters

`retVal`

## Return value

None
