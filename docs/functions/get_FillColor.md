# IStylesProvider::get\_FillColor method (uiautomationcore.h)

Specifies the fill color of an element in a document.

This property is read-only.

## Syntax

```
HRESULT get_FillColor(
  int *retVal
);
```

## Parameters

`retVal`

## Return value

None
