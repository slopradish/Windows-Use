# IUIAutomationScrollItemPattern interface (uiautomationclient.h)

Exposes a method that enables an item in a scrollable view to be placed in a visible portion of the view.

## Inheritance

The **IUIAutomationScrollItemPattern** interface inherits from the [IUnknown](/en-us/windows/desktop/api/unknwn/nn-unknwn-iunknown) interface. **IUIAutomationScrollItemPattern** also has these types of members:

## Methods

The **IUIAutomationScrollItemPattern** interface has these methods.

|  |
| --- |
| [IUIAutomationScrollItemPattern::ScrollIntoView](nf-uiautomationclient-iuiautomationscrollitempattern-scrollintoview)   Scrolls the content area of a container object to display the UI Automation element within the visible region (viewport) of the container. |
