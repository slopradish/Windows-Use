# IUIAutomationTextPattern2 interface (uiautomationclient.h)

Extends the [IUIAutomationTextPattern](/en-us/windows/desktop/api/uiautomationclient/nn-uiautomationclient-iuiautomationtextpattern) interface.

## Inheritance

The **IUIAutomationTextPattern2** interface inherits from [IUIAutomationTextPattern](/en-us/windows/desktop/api/uiautomationclient/nn-uiautomationclient-iuiautomationtextpattern). **IUIAutomationTextPattern2** also has these types of members:

## Methods

The **IUIAutomationTextPattern2** interface has these methods.

|  |
| --- |
| [IUIAutomationTextPattern2::GetCaretRange](nf-uiautomationclient-iuiautomationtextpattern2-getcaretrange)   Retrieves a zero-length text range at the location of the caret that belongs to the text-based control. |
| [IUIAutomationTextPattern2::RangeFromAnnotation](nf-uiautomationclient-iuiautomationtextpattern2-rangefromannotation)   Retrieves a text range containing the text that is the target of the annotation associated with the specified annotation element. |
