# ICoCreateLocally interface (msaatext.h)

Exposes a method that enables a client application to create a helper object in the server context.

## Inheritance

The **ICoCreateLocally** interface inherits from the [IUnknown](/en-us/windows/desktop/api/unknwn/nn-unknwn-iunknown) interface. **ICoCreateLocally** also has these types of members:

## Methods

The **ICoCreateLocally** interface has these methods.

|  |
| --- |
| [ICoCreateLocally::CoCreateLocally](nf-msaatext-icocreatelocally-cocreatelocally)   Clients call ICoCreateLocally::CoCreateLocally to create a helper object in the same context as the server object. |
