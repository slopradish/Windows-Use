# IAccIdentity interface (oleacc.h)

Exposes a method that provides a unique identifier for an accessible element.

## Inheritance

The **IAccIdentity** interface inherits from the [IUnknown](/en-us/windows/desktop/api/unknwn/nn-unknwn-iunknown) interface. **IAccIdentity** also has these types of members:

## Methods

The **IAccIdentity** interface has these methods.

|  |
| --- |
| [IAccIdentity::GetIdentityString](nf-oleacc-iaccidentity-getidentitystring)   Retrieves a string of bytes (an identity string) that uniquely identifies an accessible element. |
