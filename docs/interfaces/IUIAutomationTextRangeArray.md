# IUIAutomationTextRangeArray interface (uiautomationclient.h)

Represents a collection of [IUIAutomationTextRange](/en-us/windows/desktop/api/uiautomationclient/nn-uiautomationclient-iuiautomationtextrange) objects.

## Inheritance

The **IUIAutomationTextRangeArray** interface inherits from the [IUnknown](/en-us/windows/desktop/api/unknwn/nn-unknwn-iunknown) interface. **IUIAutomationTextRangeArray** also has these types of members:

## Methods

The **IUIAutomationTextRangeArray** interface has these methods.

|  |
| --- |
| [IUIAutomationTextRangeArray::get\_Length](nf-uiautomationclient-iuiautomationtextrangearray-get_length)   Retrieves the number of text ranges in the collection. |
| [IUIAutomationTextRangeArray::GetElement](nf-uiautomationclient-iuiautomationtextrangearray-getelement)   Retrieves a text range from the collection. |
