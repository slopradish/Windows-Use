# IUIAutomationBoolCondition interface (uiautomationclient.h)

Represents a condition that can be either **TRUE** (selects all elements) or **FALSE** (selects no elements).

## Inheritance

The **IUIAutomationBoolCondition** interface inherits from the IUIAutomationCondition interface.

## Methods

The **IUIAutomationBoolCondition** interface has these methods.

|  |
| --- |
| [IUIAutomationBoolCondition::get\_BooleanValue](nf-uiautomationclient-iuiautomationboolcondition-get_booleanvalue)   Retrieves the value of the condition:\_either TRUE or FALSE. |
