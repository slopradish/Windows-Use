# IUIAutomationCondition interface (uiautomationclient.h)

This is the primary interface for conditions used in filtering when searching for elements in the UI Automation tree. This interface has no members.

The following interfaces inherit from **IUIAutomationCondition**:

* [IUIAutomationAndCondition](/en-us/windows/desktop/api/uiautomationclient/nn-uiautomationclient-iuiautomationandcondition)
* [IUIAutomationBoolCondition](/en-us/windows/desktop/api/uiautomationclient/nn-uiautomationclient-iuiautomationboolcondition)
* [IUIAutomationNotCondition](/en-us/windows/desktop/api/uiautomationclient/nn-uiautomationclient-iuiautomationnotcondition)
* [IUIAutomationPropertyCondition](/en-us/windows/desktop/api/uiautomationclient/nn-uiautomationclient-iuiautomationpropertycondition)

## Inheritance

The **IUIAutomationCondition** interface inherits from the IUnknown interface.
