# IUIAutomationElementArray interface (uiautomationclient.h)

Represents a collection of UI Automation elements.

## Inheritance

The **IUIAutomationElementArray** interface inherits from the [IUnknown](/en-us/windows/desktop/api/unknwn/nn-unknwn-iunknown) interface. **IUIAutomationElementArray** also has these types of members:

## Methods

The **IUIAutomationElementArray** interface has these methods.

|  |
| --- |
| [IUIAutomationElementArray::get\_Length](nf-uiautomationclient-iuiautomationelementarray-get_length)   Retrieves the number of elements in the collection. (IUIAutomationElementArray.get\_Length) |
| [IUIAutomationElementArray::GetElement](nf-uiautomationclient-iuiautomationelementarray-getelement)   Retrieves a Microsoft UI Automation element from the collection. |
