# IUIAutomationNotCondition interface (uiautomationclient.h)

Represents a condition that is the negative of another condition.

## Inheritance

The **IUIAutomationNotCondition** interface inherits from [IUIAutomationCondition](/en-us/windows/desktop/api/uiautomationclient/nn-uiautomationclient-iuiautomationcondition). **IUIAutomationNotCondition** also has these types of members:

## Methods

The **IUIAutomationNotCondition** interface has these methods.

|  |
| --- |
| [IUIAutomationNotCondition::GetChild](nf-uiautomationclient-iuiautomationnotcondition-getchild)   Retrieves the condition of which this condition is the negative. |
