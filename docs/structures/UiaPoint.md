# UiaPoint structure (uiautomationcore.h)

Contains the coordinates of a point.

## Syntax

```
struct UiaPoint {
  double x;
  double y;
};
```

## Members

`x`

Type: **double**

The horizontal screen coordinate.

`y`

Type: **double**

The vertical screen coordinate.
